# foxes example: _streamline\_wakes_

This example calculates a 3 x 3 regular wind farm in a heterogeneous time series field,
with wakes that follow the streamlines of the flow.

## Check options
Check options by
```
python3 run.py -h
```

## Run command
For running on 4 cores, use
```
python3 run.py -n 4
```
